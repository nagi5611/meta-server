# 変更履歴: PDFビューワーVCポート範囲の環境変数化

## 実施日
2025年2月3日

## 概要
PDFビューワー専用の Voice Chat（VC）で使用する mediasoup の UDP ポート範囲を `.env` で指定可能にした。ルームVC（10000〜10100）とは別のポート範囲（デフォルト 20000〜20100）を使用する。

---

## 変更内容

### 1. 対象の仕様

- **PDF Viewer VC**: 同じPDFを開いている人だけが参加する音声チャット。ルームVCとは独立した mediasoup ワーカー・ルーターを使用する。
- **ポート範囲**: 環境変数 `PDF_VC_RTC_MIN_PORT` / `PDF_VC_RTC_MAX_PORT` で指定可能。未設定時は **20000〜20100** を使用。

### 2. ポート使用の前提

- ルームVCと同様、1人あたり最大 **2ポート**（送信・受信の WebRtcTransport 各1）
- ファイアウォールやクラウドのセキュリティグループでは、ここで指定した範囲の **UDP** を開放すること

### 3. 利用方法

ポート範囲を変更する場合は `.env` に以下を設定する。

```env
PDF_VC_RTC_MIN_PORT=20000
PDF_VC_RTC_MAX_PORT=20100
```

省略した場合は **20000〜20100** が使用される。

---

## 影響範囲

- 既存の `.env` に何も追加しなければデフォルト 20000〜20100 が使用される
- ルームVC（`VC_RTC_MIN_PORT` / `VC_RTC_MAX_PORT`）とは別のワーカープールのため、両方のポート範囲を開放する必要がある
